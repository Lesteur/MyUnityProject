# MyUnityProject

## Présentation

Il s'agit de mon projet personnel en Unity. C'est un Tactical RPG dans lequel vous pourrez incarner divers personnages aux gameplays différents pour affronter des monstres sur un champ de bataille.

## Progression

Le jeu est encore en grand cours de développement. J'espère pouvoir le mettre à jour, bientôt. Je me concentre sur l'efficacité, la clarté et la modulation du code.
